using ‘Tuttle’ - the Git version control extension

This chapter is about ‘Tuttle’, the Git version control extension of TEI-Publisher.

Which Git providers are supported?

git tuttle github gitlab
Tuttle supports either Github and Gitlab repositories.

Can i have multiple repositories?

git repository Github Gitlab
Yes. It’s possible to configure multiple repositories even of different types. So for instance you can have one Gitlab repository connected to one target collection in the database and another collection that is bound to a Github repository somewhere else. Each configuration item will correspond to exactly one target collection. However target collections must not be nested as that would cause trouble.
